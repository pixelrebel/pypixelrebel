### _helper_ Class

Provides common functions and snippets

###### cmpv(version1, version2)

```
Compare version strings
Returns '0' if version1 == verstion2
Returns '-1' if version1 < version2
Returns '1' if version1 > version2
```

###### commas(x)

```
Return number as string with commas
```

```

```
